# Document Review Prompt

Review the documentation in this repo snapshot or context pack.

- Look for inconsistencies, ambiguities, out-of-date claims, old terminology,
  missing risks, and confusing sections that need clarification.
- Redundancy is only worth flagging when it creates clear maintenance risk or
  duplicated conflicting guidance.
- Call out improvements as well as issues.
- Prefer concrete fixes tied to file paths.
