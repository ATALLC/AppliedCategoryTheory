# Resources

This directory collects the reading lists, videos, meetups, and working
bibliographies that support the ACT study track.

## Core Study Resources

| File | Focus |
| --- | --- |
| `books.md` | Recommended textbooks and longer-form references for ACT study. |
| `engaging-with-mathematics.md` | Study habits and practical advice for learning mathematics as an engineer. |
| `papers.md` | Core research and practitioner papers relevant to ACT in software engineering. |
| `videos.md` | Talks, lectures, playlists, and podcasts for guided study. |
| `meetups.md` | Community events, workshops, and conferences. |
| `misc.md` | Blogs, tools, threads, and other supporting references. |

## Working Bibliographies

| File | Scope |

## Notes

- Keep resource scopes explicit so readers can tell whether a page is a general
  study guide or a focused research bibliography.
